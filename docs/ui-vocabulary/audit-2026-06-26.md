## Category Decision

No category expansion is required for this milestone.

The original six buckets still match the poster-style beginner mental model: input, selection, action, structure, feedback, and data/list information. `data-display` and `structure` are now the largest buckets, but splitting them immediately would make the first navigation pass less beginner-friendly and would require a broader UX decision about whether the site should keep poster-style categories or move toward design-system taxonomy.

Recommended next-horizon category candidates:

- `navigation`: breadcrumbs, pagination, steppers, nav bars, drawers, rails, sidebars.
- `layout`: app shell, page layout, panels, rails, sticky bars, dashboard grids.
- `media-commerce`: media cards, players, gallery, product, checkout, billing, order patterns.
- `developer-settings`: API key, webhook, integration, connection, version history patterns.
